# 腊八粥
laeh8 baeh7 zhoh7
+ 仿效佛祖释迦牟尼成道前牧女献乳糜的传说故事，取香谷及果实等造粥供佛，名腊八粥。后流行于民间。据《燕京岁时记》载：“腊八粥者，用黄米、白米、江米、小米、菱角米、粟子、红豇豆、去皮枣泥等，合水煮粥，外用染红桃仁、杏仁、瓜子、花生、榛瓤、松子及白糖、红糖、葡萄以作点染。”如今的八宝粥，就是参照腊八粥的配料制作的。
  * 如东方言词典
+ 农历十二月初八日用米，豆等谷物和枣、栗、莲子等八种干果煮成的粥。起源于佛教。
  * 东台方言词典
+ 亦称佛粥，据说释迦牟尼经过六年苦修，方于腊月八日在菩提树下悟道成佛，此日遂为佛教节日；每年十二月初八日，各大寺院均用果子杂拌煮粥，分食僧众，乃有此称，后来民间相沿成俗。民俗，腊八粥必用青菜，加红枣、黄豆、蚕豆、红豆、花生米、山芋、芋头等为之，泰州光孝寺恢复重建后，每逢腊八，亦以大锅煮粥馈赠百姓。
