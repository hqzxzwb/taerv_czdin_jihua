# 量升斠斗
lian2 shen1 go5 deu3
+ 原指用斗和升来测定粮食存量，后泛指清仓查库，摸清家底。（见“斠”）
  * 东台方言词典
+ 含贬义，不放心别人；背着人查问题。
  * 东台方言词典
+ 反复点数、反复核算
  * 兴化方言词典
    - 他买点东西家来有得～唻！
+ 反复用升斗来校正是否准确，意谓因疑重量不足或者多出而反复斠秤、折腾不已
  * 泰州方言词典4
  - 你就不要再～唻，多个两把，少个两把，都没得说相，百十斤鱼在这块分吖半天还不曾分得好！
