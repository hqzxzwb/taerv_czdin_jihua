# 亮星
lian6 xien1
+ 星光或者月光很亮的晚上
+ 阴历初八至二十三之间，月亮在空中，光线明亮，称亮星。
  * 如东方言词典
+ 天空没有云雾，星光闪烁，这种天气称亮星。
  * 如东方言词典
+ 有月的夜间。一般指阴历初八至二十三之间，即使乌云密布，天色也亮，不会一片漆黑。
  * 如皋方言词典
+ 明亮的夜晚。
  * 东台方言词典
+ 有月亮：今朝是～。
  * 泰县方言志
+ 月光明亮的夜晚或星光很亮的夜晚
  * 泰州方言词典3
