# 软当/软裆
run3 dan1(rg:0)
+ 腰侧软区。
  * 如皋方言词典
+ 原指身体软弱的部位，引申指存在的薄弱环节
  * 东台方言词典
  - 通过这次检查，一定要把～弥补起来。
+ 指人身体上没有骨骼支护的部分，如腹部、胯裆等处。
  * 兴化方言词典
+ （引）弱点和缺点
  * 兴化方言词典
  - 他再凶就捣他的～。也说“软塘”。
+ 男性外生殖器部位以及其他身体表面没有骨骼支撑的部位，民间视为易遭到外来攻击的致命之处
  * 泰州方言词典3
  - 一撉儿打啊在他的～上。
+ 个人重大欠缺之处
  * 泰州方言词典3
  - 他这个人各个方面都不丑，就是学历低啊点儿，要不是这个～，老早就捱提拔当主任唻。
+ 喻弱点；存在问题的地方。
  * 泰兴方言辞典
    - 抓住他的～他就凶不起来了。
