# 喜钱
xi3 qin
> 如东方言词典
- 一块长方形的纸红色，中间雕花刻字飞金，下沿飘须。春节贴在大门、房门、耳门门楣以及照明方上，表示吉庆。如果当年有人去世，则贴紫色的喜钱，表示悼念。俗称过儿。

# 喜笺
xi3 qin/jin
+ 春节或家中有大喜事时张贴于门窗上的红色刻纸笺，表示喜庆。
  * 泰兴方言辞典

# 喜签
xi3 qin1
> 东台方言词典
- 贴春联时贴在门楣上、房梁上或者窗户上边框的一种刻纸。也叫“花签”“花钱”。

# 喜钱
xi3 qin2
+ 逢喜庆事给的赏钱，常用红纸包裹。
  * 如皋方言词典
+ 办喜事时发的红包。
  * 东台方言词典
+ 有喜庆之事时，给人的赏钱。
  * 泰兴方言辞典
