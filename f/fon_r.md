# 封儿
fon1 r
+ 举办红、自喜事时，将钱用红纸包起来赠送给办事人的一种小礼，称封儿。主要有裁缝封儿、木匠封儿、漆匠封儿、开门封儿（关门封儿）、坐席封儿、暖轿封儿、梳妆封儿（上头封儿）、开脸封儿、上灶封儿、烧锅封儿、端盘封儿、小封儿（帮杂人员劈柴的、洗菜的、洗碗的）等。
  * 如东方言词典
+ 包有礼金的纸包，按贺仪、丧仪分红白二色。
  * 兴化方言词典
+ 逢年节或初次见面时给晚辈的红纸钱包。也称“包喜”。
  * 兴化方言词典
+ 旧时指装在红封套里或用红纸包起来的赏钱，今日是指对给予自己帮助之人表示谢意之钱。今日泰州表示谢意的封儿多用信封代替，但是给在丧事期间帮忙的人送封儿必须用红纸包
  * 泰州方言词典2
  - 小的上学都亏老王帮忙，你包个封儿把人家，
  - 明吖来帮吖搬家的司机，你要一个人准备一包香烟、一个封儿。
+ 装有礼金的纸套，泰州风俗，结婚、生日等贺仪礼金用红纸封，奠仪礼金用白纸封
  * 泰州方言词典2
  - 你家外甥女出门，你做舅舅的要包个大红纸～。
