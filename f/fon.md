# 风
fon1
+ 由于气压分布不均匀，引起地面空气流动而产生风。风力通常分为十三级，即零级至十二级。
  * 如东方言词典
+ 一种腌制鸡、鱼的方法。腊月，把鸡、鱼加盐，捆扎起来，放在背阳的地方风干，味道香。
  * 如东方言词典
+ 即借助风力把混在稻麦中的芒、稳子（颖片）扬掉
  * 兴化方言词典
