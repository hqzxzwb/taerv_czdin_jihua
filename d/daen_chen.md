# 掸尘
daen3 chen2
+ 在阴历十二月二十四日以后，把屋梁、墙壁、窗户等地方的灰尘打扫干净，称掸尘。
  * 如东方言词典
+ 住宅大扫除，一般年前、喜庆、装修时进行。
  * 如皋方言词典
+ （民俗）“送灶”过后，家家大扫除，清除房梁屋角积蓄灰尘，又叫“刷尘”“掸陈”。
