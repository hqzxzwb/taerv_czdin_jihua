# 瓭子
dun1 zz
+ 衡器。竹或金属制，圆筒形，有柄。有一两、二两、半斤等多种规格。用以打酒、醋、食油、酱油等。
  * 如东方言词典
+ 一种有长柄的量具
  * 如皋方言词典
    - 酒～、油～、酱油～等。
+ 一种空心铜锤，用以生产麻油
  * 如皋方言词典
    - 用麻油～瓭麻油。
+ 旧时打油和酒的工具，用毛竹制成，后以马口铁代替，有一两、半斤、一斤不等。
  * 东台方言词典
+ 量油、酒的小圆筒，竹子或铁皮制成，有长柄。瓭，《集韵·覃韵》都含切：罂也。俗作“端子”。
  * 兴化方言词典
    - 油～；酒～
