# 戥子
den3 zz
+ 构造和原理与杆秤相同。秤杆一般用骨质材料制成，十分精致。盛物的部分是一个小圆盘。最大单位是两，小到分、钱、厘、毫。用于衡量贵重物品。
  * 如东方言词典
+ 衡量精致物品和药品重量的器具，最大单位是两，小到分或厘（老秤十六两制小到钱）。
  * 东台方言词典

# 墩子
den1 zz

  * 东台方言词典
+ 土堆
  * 兴化方言词典

  * 兴化方言词典
