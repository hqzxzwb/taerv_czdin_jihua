# 一衝之性
ieh7 chon3 zhr1 xien5
> 如皋方言词典
- 冲动、不冷静
  - 少年人～做蠢事是常事，要接受教训，学会冷静才好。

# 一冲之兴
ieh7 chon5 zhr1 xien5
> 东台方言词典
- 不假思索依着一时冲动和兴致说话、办事
  - 你～，哿晓得后果有多严重？！

# 一冲之兴
ieh7 chon1 zhr1 xien5
> 兴化方言词典
- 由着性子，凭好恶行事
  - 他做事总是～，过后就丢啊干干净净。也说“一兴之头”。

# 一冲之兴
ieh7 chon5 zhr1 xien5
> 泰兴方言辞典

  - 他～，失手打人，是不对的。
