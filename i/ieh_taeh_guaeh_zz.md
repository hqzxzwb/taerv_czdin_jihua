# 一塌刮子

+ 统统，总共。
  * 如东方言词典
    - ～有多少？
+ 总共，合计的意思。
  * 东台方言词典
+ 通通、全部
  * 兴化方言词典
    - 所有的东西～全在这块。
+ 一共、总共
  * 兴化方言词典
    - 一茶三点一个面，～五块钱不到。
+ 一共
  * 泰县方言志

+ 一古脑儿；全部，总共，所有的。
  * 泰兴方言辞典
