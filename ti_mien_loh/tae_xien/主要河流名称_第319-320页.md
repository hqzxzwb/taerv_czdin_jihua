| 名称     | 页码 | 备注 |
|----------|------|------|
| 靖泰界河 | 319  | 为靖江、泰兴两县的分界线。西起长江边，东至珊瑚公社二十号（村）。沿线流经七圩、新市、曲霞、广陵公社的地域。全长近45公里，河口宽35一70米。 |
| 如泰运河 | 319  | 由如皋西流，经我县分界（分界公社驻地）、刘堡（隶属三里公社）、黄桥镇、坍江头（隶属溪桥公社）、河头庄（河失公社驻地）、方阡（隶属姚王公社）、泰兴镇、陈家园子（隶属燕头公社）、至过船港口入江。我县境内长45公里。河口宽50一55米。 |
| 东姜黄河 | 319  | 由泰县姜堰镇南流，经我县新桥（隶属元竹公社）、西雁岭（隶属横垛公社）、碾坊头（三里公社驻地）至黄桥镇与如泰运河汇合，我县境内长近12公里。河口宽50一58米。 |
| 西姜黄河 | 319  | 由泰县姜堰镇南流，经我县霍家庄（隶属北新公社）、丁家庄（隶属元竹公社）、十里王家庄（隶属三里公社）、至黄桥镇与如泰运河汇合，我县境内长15公里。河口宽40一52米。 |
| 季黄河   | 319  | 由黄桥镇南流，经横巷（横巷公社驻地）、许王庄（隶属珊瑚公社）、至靖江县季家市入靖泰界河，全长15公里。河口宽75一80米。 |
| 两泰官河 | 319  | 由泰县南流，经我县孔家桥（孔桥公社驻地）、毛家群（隶属宣堡公社）、李秀河、渡子河（隶属根思公社）、至泰兴镇汇入如泰运河，我县境内长近23公里。河口宽40一50米。 |
| 南官河   | 319  | 旧称庙港。由通扬运河南流。经我县刁家铺、口岸至高港入江，我县境内长近14公里，河口宽60一70米. |
| 增产港   | 319  | 由古马干河南流，贯穿分界、长生二公社，至珊瑚公社十七号（村）汇入靖泰界河，全长近23公里，河口宽50米左右。 |
| 古马干河 | 319  | 由古溪西流，经钱家荡（隶属横垛公社）、丁家庄（隶属元竹公社）、南新街（南新公社驻地）、马家庄（隶属老叶公社）、李秀河（隶属根思公社）、至马甸闸入江，全长近44公里。河口宽46一57米。解放后，系利用马甸港，新庄子河、众安港、王庄河、古黄河分期截弯取直开挖而成。 |
| 焦土港   | 319  | 由珊瑚庄西流，经宁界、南沙、常周、曲霞、张桥、新市六个公社，至蒋华公社入江。全长近37公里。河口宽40一55米。解放后，系利用原和尚港、横河改道开挖而成。 |
| 新曲河   | 319  | 北起北新公社驻地倪浒庄，流经南新、刘陈二公社，至河失公社河头庄接如泰运河，全长14公里。河口宽47一50米。（该河原计划由倪浒庄开挖至曲霞公社驻地曲霞镇，故定名新曲河）。1976年开挖。 |
| 天星港   | 320  | 由季黄河西流，经钱葛巷（隶属溪桥公社）、西鹤脚湾（隶属常周公社）、杨家营（隶属焦荡公社）、常巷（隶属张桥公社）、至天星港口入江。全长近32公里。河口宽45一50米，1969年开挖而成。 |
| 羌溪河   | 320  | 旧名姜溪河，俗名将军河。由如泰运河南流经张家桥（张桥公社驻地）至毗芦市（隶属新市公社）入靖泰界河，全长近13公里。河口宽40一50米。 |
| 宣堡港   | 320  | 由西姜黄河西流，经汪群、宣堡、孔桥、田河等公社至口岸接通南官河，全长近40公里，河口宽45一50米。1976年1月开挖而成。 |
| 许庄河   | 320  | 东起两泰官河，西接南官河，全长10公里，河口宽44米，1978年新开。 |
