# 雁
ngaen6
> 如东方言词典
- 鸭科动物白额雁,又名豆雁、鸿雁,是家鹅的祖先。雄者体长约70厘米,雌者较小。嘴扁平,被有软皮,肉色或玫瑰色,尖端具角质嘴甲,灰色或白色。虹膜棕色。嘴基和前额皆有白色横纹。头、颈背部羽毛棕黑,羽缘灰白色。胸、腹部棕褐色,布有不规则黑斑。腿和脚橙黄色,有4趾,前3趾间具蹼,后1中上小而不着地,蹼淡黄色;爪短而钝白色或灰色。栖息在沼泽地区。迁徙时,集成大群,飞行有序。主食植物,间食昆虫。
