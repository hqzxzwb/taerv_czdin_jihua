# 丫头
nga1 teu
+ 女儿
  * 如皋方言词典
    + 昵称女儿。
  * 如东方言词典
  * 泰州方言词典2
  - 他家三个新妇养的都是～/都是女儿（相对于“小伙”）
+ 泛指女孩子，或称呼女孩子
  * 如皋方言词典
    + 旧时女孩在项两侧梳妑妑儿，头脑后垂长发，象“丫”字，因称女孩为～。但有感情色彩和身份。
  * 如东方言词典
  * 泰兴方言辞典
    + 旧时女孩头上常梳丫形发髻，故名。
  * 东台方言词典
+ 年轻女子
  * 如皋方言词典
    + 贬、昵称青少年女性。
  * 如东方言词典
  * 泰州方言词典2
    + 泛指青年女子，带有些许蔑义
    - 这个～竟然也当吖干部唻。
+ 丫环的俗称。
  * 如皋方言词典
  * 泰兴方言辞典
+ 长辈对小辈妇女的昵称
  * 泰兴方言辞典
    + 如贾母称风姐为“凤丫头”。
  * 泰州方言词典2
