# 开脸
kae1 lin3
+ 姑娘结婚后的第二天早晨，摘除脸上的绒毛，叫做开脸。也叫做扯脸。
  * 如东方言词典
+ 见“扯脸”。
  * 如皋方言词典
+ 旧时女子未婚时脸上有汗毛，额际有刘海儿，鬓脚，颈部有短发，称“毛丫头”。结婚，则于次日晨由妆妈用线扯去，脱去姑娘的脸型而成为大（da5）娘，妇人。
  * 如皋方言词典
+ 旧时姑娘出嫁前一天，娘家人用绞在一起的细线一张一合为女儿清除脸上汗毛。也叫“扯脸”“绞脸”。
  * 东台方言词典
+ 旧俗女子不修面，直到婚前才初次修面整鬓，叫做“开脸”。
  * 泰兴方言辞典
