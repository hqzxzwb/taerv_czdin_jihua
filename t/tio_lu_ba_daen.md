# 挑箩扒担
tio1 lu2 ba1 daen1
> 东台方言词典
- 用箩筐挑粮时，要用笆斗从船舱或粮囤上扒粮装筐。故以此泛指搬运工。

# 挑箩把担
tio1 lu2 ba3 daen5
+ （借）指用时的搬运工人
  * 兴化方言词典
    - 四个米囤，由红到白，各有不同的买主。头糙卖给～卖力气的……（汪曾祺《八千岁》）
+ 旧社会对搬运工人的称呼或出身微贱的人。
  * 泰县方言志

# 挑箩把担
tio1 lu2 ba3 daen5
> 泰州方言词典4
- 挑着箩筐、控制扁担，保持担子平衡，引伸喻指搬运职业。箩筐、扁担乃是旧时搬运工人主要工具，装卸粮食、煤炭、盐、糖等大宗物资均用箩担，故有此喻
  - 你现在不好好的用功，大啊就只好去～唻！
- 泛指处于社会下层依靠出卖力气吃饭的人
  - 你俫不要看不起这些～的，人家不偷不抢，靠卖力气吃饭，也不丢人。
