# 闼子

+ 用木板制成的活动式的组合门。

+ 上在门坎槽里的长板门
  * 兴化方言词典
    - 当三年学徒，每天早上起来下～，打扫……（兴文《刘熙载传说》）

+ 一种由多块可移动的长方形门板组合而成的商铺面街大门，门板两端嵌在商店门面上下槽（上槽固定，下槽可以移动）内，商店开门之时一块一块卸走，关门之时一块一块装上，组成板壁，板壁中间由二块有轴的门板构成可开关的小门，以其“多个小门组合而成”，故名
  * 泰州方言词典3
