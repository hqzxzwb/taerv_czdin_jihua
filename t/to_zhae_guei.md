# 讨债鬼
to3 zhae5 guei3
+ 置语或昵语。未过十三岁夭折者为～。
  * 如皋方言词典
+ 子女没出息，非但不能赡养父母，反而靠啃老生活，好像父母上辈子欠了他们一笔债似的，这就叫讨债鬼。
  * 东台方言词典
+ 子女未成年就夭折了。
  * 东台方言词典
+ 迷信认为多病、败家或夭寿的孩子是向父母索取前生债而来的～。
  * 兴化方言词典
+ 对游手好闲青年的詈词。
  * 兴化方言词典
+ 佛教以为父母因前世欠下儿女的债而未归还，所以债主投胎成为欠债者的儿女前来讨债，父母省吃俭用挣钱全部用于儿女，是还前世所欠的债，故称儿女为“讨债鬼”，稍带贬义。
  * 泰州方言词典4
+ 迷信认为，生子有三个原因，一是父母前世负子的债，二是儿子前世负了父母的债，三是对头怨家来作子。凡是所生之子早天的，便是第三种原因，故称未成年而天折者为“讨债鬼”。乡人常用于责骂劣子或反其义而亲昵地戏称小儿。
  * 泰兴方言辞典
