# 抬颏
tae2 hae6
> 兴化方言词典
- （借）好、有出息
  - 这个小伙～，个钱总舍不得瞎用。
- 称产品质量好，结实耐用
  - 这种货价码虽大，可东西～㖸！
- 指女子作风正，不苟且
  - 他家姑娘～，一点不瞎来。
- 抬颏一词源于金、元以来杂剧。《董西厢》二“穿一领绸杉……畅好台孩，举止无俗志。”《谢天香》一折“你觑他交椅上～样儿……”

# 胎谐
tae1 hae
+ 有出息，还可指事物
  * 如皋方言词典
    - 这什衣裳没款式，不～。
+ 吉利。
  * 如东方言词典
    - 他俫这吖下去会不～的。
+ 顺利。
  * 如东方言词典
    - 这件事不～。
+ 或作“胎亥”，叠韵连绵词，其意为有出息有本事守规矩等。《吴起敌秦》二姬栾白今日出阵显～，上的马去拿榜牌。
  * 东台方言词典
+ 长进，上进，多指人在学问、技艺、品行某一方面有所进步
  * 泰州方言词典3
  - 伢儿你要放吖～点儿，争取考个一百分。
  - 这小伙不～，弄吖在家里偷他家老子的好香烟吃。
