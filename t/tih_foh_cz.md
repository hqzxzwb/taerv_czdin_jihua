# 贴福字
tih7 foh7 cz6
+ 春节，在大门米柜上贴福字，表示富贵幸福。比喻说好话，恭维。
  * 如东方言词典
    - 你就会～啊！
+ ［喻］说恭维话。
  * 东台方言词典
+ 说好话、顺耳的话
  * 兴化方言词典
    - 他专会～，有话也不直说。也说“贴顺遂条儿”。
+ 恭维，奉承。
  * 泰县方言志
+ 民俗，每年春节，除夕这天，家家户户都要在大门或者门楣、墙壁，甚至家具、农具上贴上大大小小红纸斗方的“福”字，并将“福”字有意倒贴，取“福到”之意。
  * 泰州方言词典4
+ 比喻说对人有利的话
  * 泰州方言词典4
  - 市委组织部来开座谈会，征求对提拔小李的意见，上头定下来的，大家还有甚的说的，个个～。
+ 说中听的话；喻溜须拍马。
  * 泰兴方言辞典
