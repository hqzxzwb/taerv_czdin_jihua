# 挂红
gua5 hon2
+ 驱邪风俗
  * 东台方言词典
    + 在门或某物上挂一块红布，同时敬香放爆竹，以消灾袪邪，图个吉利。有迷信色彩。
  * 兴化方言词典
    + 有婴幼儿出天花，一方婴幼儿衭头均挂红布条，上书“天行已过。”
  * 泰州方言词典2
    + 旧俗，如有婴幼儿出天花，以为瘟疫所至，附近人家为防家中小儿传染，在小儿衣帽上挂红布条，上书“天行已过”，意思是流行瘟病已经结束。“天行”指流行传染病
    - 这个庄上伢儿身上挂啊红，肯定是有人得咖天花。
+ 檐挂红布或红绸，表示庆贺
  * 兴化方言词典
    - 到啊儿子结婚，家里～结彩，热烈庆贺。
    - 王家遭回禄后，四邻均在檐口～，庆幸逃过一劫。
+ （戏）面部流血
  * 兴化方言词典
    - “……我又没甚喜事，怎么嘴上挂了红耶？”（《西游记》三二）
