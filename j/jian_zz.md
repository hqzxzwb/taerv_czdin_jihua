# 糨子
jian5 zz
+ 用糨糊把几层布粘成板块，晒干，这种材料叫做糨子。主要用于做布鞋的鞋底和鞋帮。
  * 如东方言词典
+ 将旧、碎布用薄浆糊糊几层，晒干后的成品，做布鞋用。家中可自制，专糊糨子的人家也有售。
  * 如皋方言词典
+ 糨糊，用面粉等做成的可以粘贴东西的糊状物。
  * 东台方言词典
+ 用破布一层一层铺平糊成，专用缝制布鞋底的东西。
  * 泰县方言志
