# 叫魂
jio5 huen2
+ 一种玄学仪式，使人魂魄归体。也引申詈人叫唤。
  * 如东方言词典
    + 一种迷信活动。人溺水死亡后，灵魂走散，通过叫魂，把魂灵叫回来。
    - 我才离开，你就～。
  * 如皋方言词典
    + 旧俗称人受惊吓则魂魄失落而致病，把魂叫回来病就好了。
  * 东台方言词典
    + 迷信认为人患病是因灵魂出窍所致，呼唤病人的名字可以唤回灵魂，治好疾病。
    - 你不要在外头～，家去再说。
  * 兴化方言词典
    + 呼唤受惊患儿的名字，使灵魂回到身上。
    - 你弄吖这块～，烦死人吖！
  * 泰州方言词典2
    + 旧俗，以为儿童摔跤跌倒，会导致魂魄离体，家人扶起儿童以后，一遍用手指蘸地上土，点在儿童鼻尖，一边呼唤儿童名字，据说可以使儿童魂魄归体。
    - 有人在楼底下～喊你，你快点儿下去。
  * 泰兴方言辞典
    + 也称“喊魂”。旧时迷信，以人受惊吓而病或病危，是魂魄离体所致，召唤使之附体谓之叫魂。引申为责人大声喊叫为叫魂。
    - 天天～吖似的，让人烦煞咖。
