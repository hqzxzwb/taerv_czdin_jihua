# 五红
v3 hon2
+ 民俗，端午五样红色菜肴
  * 如皋方言词典
    + 如红烧肉、红萝卜、苋菜、火腿、虾等。（另有五绿、五黄同为端午菜肴）
  * 东台方言词典
    + 端午节午餐中的红萝卜、盐水虾、红苋菜、咸鸭蛋、红烧鱼，也可加上红烧肉、西红柿等红色食品任选五样。据说吃五红可去“五毒”。
  * 兴化方言词典
    + 端午节的午餐中要有炒虾、红烧肉、红煮鱼、苋菜、咸鸭蛋或红萝卜等五样红色的菜肴。
  * 泰兴方言辞典
    + 端午节中午家家吃“五红”。一般为红烧肉、红烧鱼、虾、红苋菜、咸鸭蛋，谓可避邪、保健。
