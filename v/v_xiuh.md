# 焐血
v5 xiuh7
+ 指人摔伤或受钝器撞击后，受伤部位的血肿
  * 兴化方言词典
  - 额头上倒撞吖～咖。
+ 身体受到外力打击以后，受伤部位形成血肿但未破皮
  * 泰州方言词典3
  - 这小伙打起来没得数，把伢儿身上打啊几处～。
  - 他这一下子跌得不轻，小膀子上都焐啊血。

# 焐雪
v5 xiuh7
+ 天气阴冷，云层𨠒红，预示高空酝酿下雪。
  * 如皋方言词典
+ 见“作雪”。
  * 如皋方言词典
+ 冬季天气阴沉，即将下雪
  * 兴化方言词典
    - 天上的云这吖厚，又没风，怕的～咖。
+ 指冬天天气阴沉而又略感暖意，预示天要下雪
  * 泰州方言词典3
  - 下午天阴吖来啊，一点儿不觉冷，望这样子像是要～。
