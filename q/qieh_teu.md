# 七头
qieh7 teu2
+ 死者去世举办丧事，从发丧至脱丧之前，死者儿子、孙子不得理发。儿孙之中如在死者去世之前数日已经理发的，一般不再理发；如有在死者去世之前应理发而未理发的，一般均在死者去世当日去理发店理发，称为“剃七头”。
  * 泰州方言词典3
+ 詈词，骂男人不理发导致头发过长。
  * 泰州方言词典3
