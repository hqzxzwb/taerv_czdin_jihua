# 喊魂
haen5 huen2
+ 迷信者认为，小儿受惊后，魂灵吓跑了，大人夜间呼喊其名，招魂回归，这种活动名为喊魂。也叫招魂。
  * 如东方言词典
+ 大声地不断地呼喊。
  * 如东方言词典
    - 你不要～吖。
+ 见“叫魂”。
  * 如皋方言词典
+ 詈语，指无故大喊大叫。
  * 如皋方言词典

# 喊魂
haen3 huen2
+ 小孩受惊后生了病，迷信说法是魂掉了，要到受惊的地方把魂喊回来。现多指声音太大令人讨厌
  * 东台方言词典
    - 人家都在睡中觉，你在外头喊甚的魂吖？
+ 为受惊的孩子把魂。
  * 兴化方言词典
+ 骂人大喊大叫：覅～吧，烦死啊。‖也称“叫魂”。
  * 兴化方言词典
+ 小孩受惊，大人呼喊招魂。
  * 泰县方言志
+ 大声呼喊：你不要～，难听煞咖。
