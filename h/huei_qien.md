# 悔亲
huei3 qien1
> 东台方言词典
- 又叫“退亲”。意思男女双方订婚后，其中一方由于某种原因，提出解除婚约。

# 会亲
huei6 qien1
+ 姑娘结婚后第三天，娘家人到婆家去认识亲戚，叫做会亲；第四天，婆家人到娘家去认识亲戚，叫做反会亲。也称双会亲。参加会亲的人，在新娘生小孩时都要送产妇礼。
  * 如东方言词典
+ 婚仪之一。结婚后，男家邀女家亲族赴宴，与男家亲族互相承认亲戚关系。赴宴者“不空手”须带礼物去，又“回半礼”带回。
  * 如皋方言词典
+ 结婚时男女双方各自置办宴席，请对方亲友见面。
  * 东台方言词典
+ 男女婚姻关系确立后，男方邀集双方主要亲戚聚会认亲
  * 兴化方言词典
    - 新人上轿，冯老孺人也上轿，送到金家，……次日，杨家哥嫂多来～……（《二刻》九）

# 会亲
huei6 qien1
> 泰州方言词典2
- 旧时传统婚礼喜日婚宴均由男方举办，女方父母、兄弟、姐妹以及相关亲戚都会回避。男方乃在婚后数日设下酒筵，宴请女方父母、兄弟、姐妹以及相关亲戚，双方互相“认亲”，称为会亲。《民国泰县志稿》卷之23:“会亲，婚后数日，女之伯叔兄弟临男宅视女曰看亲。男宅择日请岳家眷属及做朝之亲族，曰会亲。”
