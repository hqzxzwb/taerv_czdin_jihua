# 筹钱
cheu2 qin2
+ 车水计数筹码。铜钱串在一根绳索上，水车练旋转一圈丢一枚铜钱，称一筹。赛车水时，一根绳上有40筹（39枚铜钱）、60筹（59枚铜钱），不车赛水时，一根绳上有50筹（49枚铜钱）、100筹（99枚铜钱）两种。
  * 如东方言词典
+ 一串铜钱，有39、59（赛水）和49、99（平时）枚四种，为踏水计数器。踏水时，沷鹤儿在槽内转一周称“一筹水”，拨一铜钱另一端。拨尽为“一期水”，又称“一范水”。
  * 如皋方言词典


# 抽签
cheu1 qin1
> 泰州方言词典2
- 指人遇到事情犹豫不决，难下决策
  - 你到底是去，还是不去，不要～。
- 事物发展过程变化无常，始终没有结果
  - 他本来答应帮忙的，后来说没得工夫，只好请老王，老王又说家里有事，你说哿～吖。
