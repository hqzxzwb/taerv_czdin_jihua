# 字儿镘儿
cz6 r mun6 r
> 泰州方言词典4
- 此语乃是旧时玩“跌博”游戏的术语，“跌博”之时，将六枚钱同时扔在地上，地上的钱全为字者、全为镘者为胜，钱在滚动之时，跌博之人或者口呼“字儿”，或者口呼“镘儿”。
- 指事情结果好坏
  - 你家小夫这回子考是考过啊，不过～还不晓得怎吖说吤。
- 指说话的不确定性
  - 你是去，还是不去，干脆点儿，我还有其他事，没得工夫同你～的。
