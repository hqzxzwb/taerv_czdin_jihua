# 撑硬头篙
chen1 ngen6 teu2 go1
> 泰州方言词典4
- 比喻在不具备做成某事条件或者实力的情况下，极勉强的去做某事而不愿意认输。竹篙用竹做成，一根竹子两头，竹子顶部的一头细而软，是为“软头”，竹子的根部一头硬而粗，是为“硬头”，船夫撑船之时，均为手握“软头”，将“硬头”抵住河底使船前进非常容易，是为“撑软头篙”，反之，手握“硬头”，将“软头”抵住河底使船前进就很困难，是为“撑硬头篙”
  - 他拿多少钱，你也拿多少钱，你就不想想自己拿啊几个工资，你拼不过他的，就不要撑硬头篙味。
  - 二百斤你举不上去的，不要撑硬头篙唻，把自己弄吖伤吖不上算。
