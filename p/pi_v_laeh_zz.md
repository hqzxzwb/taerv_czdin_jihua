# 皮五辣子
pi2 v3 laeh8 zz
+ 见“皮老五”。
  * 如皋方言词典
+ 小说《清风闸》的主人公，一个谁也惹不起的泼皮无赖。
  * 东台方言词典
+ 戏称顽皮小儿。清浦琳撰扬州评话《清风闸》中的主人翁。皮市井无赖，绰号皮五辣子，日久人皆不知其名。本故事在苏皖地区流传甚广。《扬州画航录》卷9《小秦淮录》载：“浦琳，字天玉，……乃以己历之境，假名皮五，撰为《清风闸》故事。”是书脍炙人口，阅者无不称羡之。

# 皮五瘌子
pi2 v3 laeh8 zz
> 泰州方言词典4
- 小说《清风闸》的主人公皮奉山，排行第五，因为头生恶疮而秃，人称“皮五瘌子”，皮奉山未发迹时是个地痞光棍，一旦身上无钱赌博之时，就会想方设法放刁耍赖讹人钱物。后来遂以“皮五瘌子”指想方设法放刁耍赖讹人的人
  - 他是个～，你惹上吖甩都甩不掉。
