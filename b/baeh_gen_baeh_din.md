# 八更八点
baeh7 gen1 baeh7 din3
+ 旧时一夜分为五更，八更是没有的。表示深夜。
  * 如东方言词典
    - 哪怕～也要赶到！（形容决心很大）
+ 称极夜深。你不要担心，今朝哪怕到～总做起来。
  * 如皋方言词典
+ 一夜本来只有五更天，八更的意思是无论拖到何时，当夜也要完成某项任务。
  * 东台方言词典
+ 极言不管到什么时候
  * 兴化方言词典
    - “下次他回来，就说我找他，哪怕夜里～。”（毕飞宇《慌乱的指头》）
+ 喻夜上很晚很晚的时候。
  * 泰兴方言辞典
    - 今朝哪怕到～，我佼也要完成箇个任务。
+ 极言夜里时间之晚，带有“无论怎么晚也要怎样”的意味
  * 泰州方言词典4
  - 明朝人家客户就要来取货，你今朝晚上～都要同我弄吖好啊。
  - 你今朝夜里就是～也要赶家来。
