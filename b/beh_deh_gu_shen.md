# 不得过身
beh7 deh7 gu5 shen1
+ 不好通过。比喻日子难过。
  * 如东方言词典
    - 旧社会碰上春荒，许多人～。
+ 不能够再承受，引申为“完蛋了，不行了”
  * 如东方言词典
+ 不能解脱。
  * 如皋方言词典
+ 无法脱身。
  * 东台方言词典
+ 进退两难：他偷啊人家的东西，这回～吖。
  * 泰县方言志
+ 比喻无法逃避对罪行的惩罚或者对过错的追究
  * 泰州方言词典4
  - 只要进吖纪委，他不交代就～。
+ 比喻无法维持最基本的生存状态
  * 泰州方言词典4
  - 你跟他借钱，他自己倒～唻，哪块有个钱借把你。
  - 这个订单不弄到手，我俫厂里就要停产，真的～唻。
