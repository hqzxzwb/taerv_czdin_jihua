# 不胎谐
beh7 tae1 hae(tx,txe:2)
+ 不会勤俭持家，随便花钱。
  * 如东方言词典
    - 箇两个人～。
+ 不正经。
  * 如东方言词典
    - 这个人～。
+ 不安分
  * 如皋方言词典
    - 这小伙成天的在外头汎，～。
+ 素质差，无作为
  * 如皋方言词典
    - 他家几个伢儿总稚，做事～。
+ 不旺盛
  * 如皋方言词典
    - 这棵树搬到这块，纵活也～。如皋词典写“不诒谐”。
+ 不学好；没出息；不争气。
  * 东台方言词典
  * 泰县方言志
    - 这个家伙～。
  * 泰兴方言辞典
+ 不好
  * 泰兴方言辞典
    - 今年麦长得不～，年成不好。
