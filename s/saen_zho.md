# 三朝
saen1 zho1
+ 结婚或生育的第三天。
  * 如皋方言词典
+ 指新婚后的第三天。旧俗这一天新妇回娘家。
  * 东台方言词典
+ 婴儿出生后第三天，旧俗这一天为婴儿洗澡所举行的仪礼，称“洗三”。
  * 东台方言词典
+ 新婚的第三日。
  * 兴化方言词典
+ 婴儿出生的第三天。
  * 兴化方言词典
+ 旧俗指婚后或产后的第三天。乡间婚俗男方连续三日热闹称“大三朝”，一天内办完喜事为“一天圆”；产后第三日亦称三朝，娘家为女儿及新生儿送礼称“洗三”。
  * 泰兴方言辞典
