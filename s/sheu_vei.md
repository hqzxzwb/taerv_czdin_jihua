



# 首尾/手尾
sheu3 vei
> 兴化方言词典
- 手续、事宜
  - 我小儿子不曾结婚，还有一桩～不曾了。也作“手尾”
  - 《二十年目睹之怪现状》第四十六回有“我们……彼此没了手尾”句。

# 首尾
sheu3 vei3
+ 引指家庭长辈的责任、义务：把子女的婚事料理完，他的～也尽吖。
  * 如皋方言词典
+ 往来、勾结
  * 兴化方言词典
    - “原来县吏都是与西门庆有～的……”（《水浒传》二六）
+ 头绪
  * 兴化方言词典
    - ……运司汪承爵到任也才三个月，都不知这事的～……（《梼杌闲评》三九）
+ 苟且行为
  * 兴化方言词典
    - “……你既与那妇人没甚～，却如何与他同行同宿……”（《京本通俗小说·错斩崔宁》）
