# 上梁
shan6 lian2
+ 把桁条架到房柱上去。
  * 如东方言词典
+ 建房中架大梁，民间通常要选吉日良辰，有庆贺仪式。
  * 东台方言词典
+ 民间建房习俗，将桁条架上山墙，在中梁上挂红（绸、布）、贴福字，就位时，要放鞭炮，散糕点和钱币，说四言八句的吉利话。
  * 兴化方言词典
+ 民俗。建新房时选良辰吉日架设正梁称“上梁”，是日亲朋好友前来祝贺。
  * 泰兴方言辞典
+ 安装大梁，建房的重要工序。民俗，民间建房，中梁就位之时，上面悬挂红布、红绸，燃放爆竹，撒散点红馒头、钱币，工匠要说一些大吉大利“鸽子”。
