# 上灯
shan6 den1
+ 灯节第一天：“（正月）十三～，十四看灯，十五元宵，十六当荤，十七财神，十八落灯……”旧时灯节以点“天灯”为标志，此日开始在场头高竿上点“天灯”，可至正月十八“小落灯”，可至二月初二“大落灯”。
  * 如皋方言词典
+ 见“点灯”。
  * 如皋方言词典
+ 阴历正月十三日晚，用圆子供神供祖。庙桅上张挂灯笼，称上灯。民谣：“上灯圆子落灯面。”

  * 泰兴方言辞典
+ 农历正月十三日，旧时从这天晚上开始，普通百姓家庭以及商号开始在门前悬挂灯笼，直至正月十八日结束，故称此日为“上灯”。民俗，正月十三这天晚上家家吃炒圆子（用糯米粉做成的直径1至2厘米的实心圆子，锅中放少许油，炒熟装盘后蘸糖吃）后，携老小带着各种灯笼在街道上游玩。
