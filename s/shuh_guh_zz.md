# 说詥子
shuh7 guh zz
> 如皋方言词典
- 俗用“说鸽子”。
- 说顺遂、祝颂的韵语。
- 贬指说不顺遂的话。

# 说鸽子
shuh7 guh7 zz
+ 上梁时请人讲吉利话，谓“请人说鸽子”
+ 玩笑式地说某人好话，旁人谓之“同某人说鸽子”
+ ［同音］在办喜事场合说成套的贺喜顺口溜。（也作“说穀子”，因穀有吉利、善、好之义）
  * 东台方言词典
+ 民间建房上梁或新人拜堂时，说的四言八句的吉利话。
  * 兴化方言词典
+ 民间举行建房上梁、新船下水、新人拜堂、灵柩下葬等仪式时所说的吉利话，多为随时即兴的顺口溜。说鸽子者多为兼职之人，建房上梁、新船下水多为木匠，新人拜堂多为主持婚礼司仪，灵柩下葬多为土工。
