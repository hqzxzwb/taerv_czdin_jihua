# 水母娘娘
shuei3 mu3 nian2 nian
+ 如皋民间故事中蝎子精的母亲。因危害百姓，观音菩萨变化除害，使她吃了幻化后的面条，被铁索贯肠，锁在通州（今南通）北门的井中，允许蝎子精一年一次看妈妈。见“蝎子精”条。
  * 如皋方言词典
+ 传说中的女性水神，旧时坊间所谓江淮地区洪水灾害之源。据元末明初陶宗仪考证，“水母”就是古代传说中的淮水水怪“无支祁”
  * 泰州方言词典4
  - 听吖老人家说，水母娘娘蹾的井就在泰州北头桥赵公桥南边的省抗排站里栋，她有三个儿子，九头鸟是她的二儿子。

# 水母娘娘哭兽头，蘣模蘣样亲乖乖
shuei3 mu3 nian2 nian2 koh7 sheu5 teu2 ton5 mu2 ton5 ian6 qien1 guae1 guae1
> 东台方言词典
- 本指水神的母亲偏爱自己的孩子，泛指通常父母亲都疼爱自己的、哪怕长相丑陋的孩子。
