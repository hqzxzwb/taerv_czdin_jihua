# 骚蜞
so1 qi2
> 如东方言词典
- 方蟹科动物无齿相手蟹，全体被坚硬的甲壳。头胸甲长约3厘米，四方形青色，少数淡红色。眼具短柄能活动。侧缘具光滑隆线，无齿。螯足1对，步足4对，密生长短不等的硬刚毛。腹部退化雄者呈三角形，雌者圆形。生活在河流泥滩上，穴居河岸、田埂。
