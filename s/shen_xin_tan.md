# 神仙汤
shen2 xin1 tan1
+ 用调味品冲开水做成的过口汤。
  * 如皋方言词典
+ 一种简单的汤菜。在碗里放入少许虾皮或紫菜、酱油、葱花、味精，用开水一冲即成。
  * 如东方言词典
+ 旧时用麻油、酱油、醋等佐料加开水冲的汤，以此下饭。
  * 东台方言词典
+ 用酱油、醋、麻油（或荤油）、小胡椒、味精等调料及蒜泥（花）冲的汤。也称“醋酸汤”“酱油汤”。
  * 兴化方言词典
+ 酱油汤。
  * 泰县方言志

# 神仙汤
shen2 xin tan1
> 泰州方言词典4
- 一种当菜下饭的汤，将酱油、胡椒或辣椒、麻油或脂油、蒜花或者葱花、味精、虾皮等作料放入碗中，冲进开水即成；神仙汤的作料根据家庭库存，并不一定齐全，最简单的制法是酱油加开水，以其最基本的原料酱油是由“神仙水”所制成，故名。
