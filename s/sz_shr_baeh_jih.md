# 四时八节
sz5 shr2 baeh7 jih7
+ 四时，四季。八节，两分两至四立。
  * 如皋方言词典
+ 略称主要传统节日。现在稍重视的大节如春节、元宵节、清明节、端午节、七月半、中秋节、重阳节、冬至节。
  * 如皋方言词典
+ 泛指一年四季的各个节刻，有时也特指按民俗有纪念，庆祝和祭祀活动的节日。
  * 东台方言词典
+ 春、夏、秋、冬四季中的立春、春分、立夏、夏至、立秋、秋分、立冬、冬至八个节日。
  * 泰州方言词典4
+ 泛指一年四季中的各个节日
  * 泰州方言词典4

+ 四时指春夏秋冬，八节指立春、春分、立夏、夏至、立秋、秋分、立冬、冬至。乡人则谓一年中的各个节气。
