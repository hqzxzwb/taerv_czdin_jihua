# 烧七
sho1 qieh7
+ 人死后，每过七天举行一次祭祀活动，称“烧七”。共举行七次，合49天。第一次，称“头七”，第二次称“二七”，最后一次，称“断七”或“终七”。
  * 如东方言词典
+ 亲人亡故之后，每隔七日一期诵经活动，名为～。“六七”进行大祭活动，直至“终七”结束。
  * 东台方言词典
+ 发怨气的口头禅。
  * 东台方言词典
+ 人死后每七天祭奠一次，是为“七”，“七七”为断七，亦称“终七”。逢“七”家人至城隍庙或土地庙带亡魂回家享祭，谓之～。（见《兴化市志，民俗》）
  * 兴化方言词典
+ 民俗，从人死当天起至第七日为“头七”，依次至“七七”。四十九天期间逢第七日，家中设立死者牌位（今以遗像代替），设斋祭奠，焚烧纸钱，故名。
