# 受记
sheu6 ji5
> 兴化方言词典
- 旧指婴儿因被孕妇抱了而引起腹泻
  - 原来有这个俗忌，大凡怀胎的抱了孩子家，那孩子就坏了脾胃，要出青粪，谓之～，直到产后方痊。（《警世通言》二五）

# 受记
sheu6 ji5
+ 迷信说法，坊间以为孕妇不可以抱别人家婴儿，如果抱了就会导致婴儿腹泻不止
  * 泰州方言词典3
  - 大肚子不能抱人家伢儿，弄吖～不得了。
+ 旧俗，婴儿成长期间，多病啼哭不止，送至佛寺接受记莂，寄名出家，坊间以为如此就可保佑婴儿健康成长
  * 泰州方言词典3
  - 把伢儿送吖庙里去～。
