# 站水碗
zhaen5 shuei3 un3
+ 迷信行为。小孩生病，疑冲撞了某神、鬼，用三根筷子直立水碗中，一边往上浇水一边祷祝，并报神鬼名讳，至筷能自立，即是撞了某神鬼，然后点香烛，烧纸镖，礼送。
  * 如皋方言词典
+ 迷信活动：（秦氏）将三根筷子插在水（碗）里，口里请着各位祖宗，提着那一位，如筷子端然不倒，便就是那一位作祟，名字叫做～。（《广陵潮》三）
  * 兴化方言词典
+ 旧时坊间流行的一种带有巫术意味的迷信风俗。旧时儿童发生头疼、发烧现象以后，家长就将大碗装满清水，数根筷子团在一起竖在水中，口里呼着亡故先人，如果呼到其中一位之时，筷子站在水中不倒，就可确定就是这位亡故先人鬼魂夜间回家探视之时，曾经用手摸过小孩头部，于是就给这位先人专门烧送纸钱，请求保佑儿童尽快痊愈。站水碗时，忌讳踢翻水碗，据说出现踢翻水碗表示家庭将要发生灾难。
  * 泰州方言词典4
+ 一种迷信活动。小孩头疼，以三根筷子站于水碗中，口念亡人之称呼，以手捞水于筷子顶，如筷子站着不倒，便是某某亡人作祟，据说焚化纸钱后小儿即安。
  * 泰兴方言辞典
